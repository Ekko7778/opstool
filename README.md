# OPSTOOL

通用运维脚本工具箱，一行命令部署。

## 一键安装

```bash
curl -fsSL https://raw.githubusercontent.com/Ekko7778/opstool/main/install.sh | bash
```

## 卸载

```bash
curl -fsSL https://raw.githubusercontent.com/Ekko7778/opstool/main/install.sh | bash -s -- uninstall
```

## 使用

```bash
ot  # 进入主菜单
```

## 支持系统

- Debian / Ubuntu
